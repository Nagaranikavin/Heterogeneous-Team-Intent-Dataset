# Folder StructureThis document describes the organization of the repository and the contents of each folder.---## Repository Structure```Heterogeneous-Team-Intent-Dataset/│├── README.md├── LICENSE│├── sample_dataset/│   ├── raw/│   ├── node_features/│   └── edge_features/│├── metadata/│   ├── feature_description.csv│   ├── mission_labels.csv│   └── scenario_description.csv│├── videos/│   ├── SSS.mp4│   ├── EPS.mp4│   ├── ASS.mp4│   └── Recon.mp4│└── docs/    ├── dataset_generation.md    └── folder_structure.md```---## Folder Description### `sample_dataset/`Contains representative sample scenarios from the complete dataset. Each sample preserves the original data format and organization used throughout the complete dataset.This folder is divided into three subfolders:- **raw/** – Original aircraft trajectory data generated by the SCRIMMAGE simulator.- **node_features/** – Processed node features describing the state and motion characteristics of individual aircraft.- **edge_features/** – Processed edge features representing pairwise interactions between aircraft.---### `metadata/`Contains supporting information required to understand and interpret the dataset.Files include:- **feature_description.csv** – Description of all node and edge features.- **mission_labels.csv** – Mission labels, mission names, and mission descriptions.- **scenario_description.csv** – Description of each simulation scenario included in the dataset.---### `videos/`Contains demonstration videos illustrating the execution of each mission type.The videos provide a visual understanding of coordinated agent behaviors and tactical role execution.Included videos:- Sky Sentinel Strike (SSS)- Escort Penetration Strike (EPS)- Anti-Ship Strike (ASS)- Reconnaissance (Recon)---### `docs/`Contains additional documentation describing the dataset.Files include:- **dataset_generation.md** – Explains how the dataset was generated using the SCRIMMAGE simulator.- **folder_structure.md** – Describes the repository organization and folder contents.---## Repository NotesOnly representative sample scenarios are included in this repository to maintain a lightweight and easily accessible dataset. The samples preserve the original file format and feature organization used throughout the complete dataset while providing sufficient information for understanding the dataset structure and reproducing the reported experiments.
